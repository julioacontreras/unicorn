# Unicorn

### What is this

[Nuxtjs](https://nuxtjs.org/) components using [Vue](https://vuejs.org/v2/guide/) and [TailwindCSS](https://tailwindcss.com/)


### Version

* Alpha 0.0.2

### How to use

Download this repository and copy in your NuctJs project `./components`.
